# Feature 3: 远程命令 (Remote Commands)

## 1. 功能概述

远程命令功能允许管理员向设备发送即时指令，执行特定操作，如重启设备、锁定屏幕、清除数据、安装应用等。支持单个设备和批量设备操作。

### 核心功能
- 即时执行远程命令
- 批量设备命令执行
- 命令执行状态追踪
- 命令历史记录
- 支持多种命令类型（重启、锁定、擦除、响铃等）
- 命令队列管理

### 支持的命令类型
| 命令 | 描述 | 权限要求 |
|------|------|----------|
| REBOOT | 重启设备 | 普通管理员 |
| LOCK | 锁定屏幕 | 普通管理员 |
| FACTORY_RESET | 恢复出厂设置 | 超级管理员 |
| WIPE_DATA | 擦除数据 | 超级管理员 |
| RING | 播放响铃（查找设备） | 普通管理员 |
| INSTALL_APP | 安装应用 | 普通管理员 |
| UNINSTALL_APP | 卸载应用 | 普通管理员 |
| RUN_SCRIPT | 执行脚本 | 超级管理员 |
| UPDATE_CONFIGURATION | 更新配置 | 普通管理员 |
| REQUEST_INFO | 请求设备信息 | 普通管理员 |

## 3. 数据库设计

### 3.1 远程命令表 (remote_commands)

```sql
CREATE TABLE remote_commands (
    id SERIAL PRIMARY KEY,
    customer_id INTEGER NOT NULL,

    -- 命令信息
    command_type VARCHAR(50) NOT NULL,
    command_payload TEXT, -- JSON格式参数

    -- 目标设备
    device_id INTEGER REFERENCES devices(id) ON DELETE CASCADE,
    -- 批量命令时设备列表
    device_ids TEXT, -- JSON数组

    -- 执行状态: PENDING, SENT, EXECUTING, COMPLETED, FAILED, CANCELLED
    status VARCHAR(20) DEFAULT 'PENDING',

    -- 时间戳
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    sent_at TIMESTAMP,
    executed_at TIMESTAMP,
    completed_at TIMESTAMP,

    -- 执行结果
    result_code INTEGER,
    result_message TEXT,

    -- 发起者
    created_by INTEGER REFERENCES users(id),

    -- 过期时间
    expires_at TIMESTAMP,

    -- 优先级: LOW, NORMAL, HIGH, CRITICAL
    priority VARCHAR(10) DEFAULT 'NORMAL',

    INDEX idx_device_status (device_id, status),
    INDEX idx_created_at (created_at),
    INDEX idx_status (status)
);
```

### 3.2 命令执行日志表 (remote_command_logs)

```sql
CREATE TABLE remote_command_logs (
    id SERIAL PRIMARY KEY,
    command_id INTEGER REFERENCES remote_commands(id) ON DELETE CASCADE,
    device_id INTEGER REFERENCES devices(id) ON DELETE CASCADE,

    -- 日志级别: INFO, WARNING, ERROR
    log_level VARCHAR(20) DEFAULT 'INFO',

    -- 日志内容
    log_message TEXT NOT NULL,

    -- 时间戳
    logged_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    INDEX idx_command (command_id),
    INDEX idx_device_time (device_id, logged_at)
);
```

### 6.2 RemoteCommandDAO

```java
@Singleton
public class RemoteCommandDAO {
    private final RemoteCommandMapper commandMapper;
    private final PushService pushService;

    @Inject
    public RemoteCommandDAO(
        RemoteCommandMapper commandMapper,
        PushService pushService
    ) {
        this.commandMapper = commandMapper;
        this.pushService = pushService;
    }

    @Transactional
    public RemoteCommand sendCommand(RemoteCommand command) {
        User currentUser = SecurityContext.get().getCurrentUser().get();
        command.setCustomerId(currentUser.getCustomerId());
        command.setCreatedBy(currentUser.getId());
        command.setStatus("PENDING");

        // 计算过期时间
        if (command.getExpiresAt() == null) {
            Calendar cal = Calendar.getInstance();
            cal.add(Calendar.HOUR, 24);
            command.setExpiresAt(cal.getTimeInMillis());
        }

        commandMapper.insertCommand(command);

        // 立即发送命令
        sendCommandToDevice(command);

        return command;
    }

    @Transactional
    public BulkCommandResult sendBulkCommand(BulkRemoteCommandRequest request) {
        List<Integer> deviceIds = request.getDeviceIds();
        BulkCommandResult result = new BulkCommandResult();

        for (Integer deviceId : deviceIds) {
            try {
                RemoteCommand command = new RemoteCommand();
                command.setCommandType(request.getCommandType());
                command.setCommandPayload(request.getCommandPayload());
                command.setDeviceId(deviceId);
                command.setPriority(request.getPriority());

                RemoteCommand saved = sendCommand(command);
                result.addSuccess(deviceId, saved.getId());
            } catch (Exception e) {
                result.addFailure(deviceId, e.getMessage());
            }
        }

        return result;
    }

    private void sendCommandToDevice(RemoteCommand command) {
        // 构建推送消息
        ObjectNode payload = JsonMapper.createObjectNode();
        payload.put("commandId", command.getId());
        payload.put("commandType", command.getCommandType());
        payload.put("payload", command.getCommandPayload());
        payload.put("timestamp", System.currentTimeMillis());

        boolean sent = pushService.sendMessage(
            command.getDeviceId(),
            PushMessage.TYPE_REMOTE_COMMAND,
            payload.toString()
        );

        // 更新状态
        if (sent) {
            commandMapper.updateCommandStatus(
                command.getId(), "SENT", null, null
            );
        } else {
            commandMapper.updateCommandStatus(
                command.getId(), "FAILED", -1, "Failed to send push notification"
            );
        }
    }

    public void updateCommandStatus(Integer commandId, String status,
                                    Integer resultCode, String resultMessage) {
        commandMapper.updateCommandStatus(commandId, status, resultCode, resultMessage);
    }

    public List<RemoteCommand> getPendingCommands(Integer deviceId) {
        return commandMapper.getPendingCommands(deviceId);
    }
}
```

### 6.3 Android Agent端命令处理

```java
// RemoteCommandReceiver.java
public class RemoteCommandReceiver extends BroadcastReceiver {
    @Override
    public void onReceive(Context context, Intent intent) {
        String messageType = intent.getStringExtra("messageType");
        if (PushMessage.TYPE_REMOTE_COMMAND.equals(messageType)) {
            String payload = intent.getStringExtra("payload");
            handleRemoteCommand(context, payload);
        }
    }

    private void handleRemoteCommand(Context context, String payload) {
        try {
            JSONObject commandJson = new JSONObject(payload);
            String commandId = commandJson.getString("commandId");
            String commandType = commandJson.getString("commandType");
            JSONObject commandPayload = commandJson.getJSONObject("payload");

            // 上报命令已接收
            reportCommandStatus(commandId, "EXECUTING", null, null);

            // 执行命令
            CommandResult result = executeCommand(context, commandType, commandPayload);

            // 上报执行结果
            reportCommandStatus(
                commandId,
                result.isSuccess() ? "COMPLETED" : "FAILED",
                result.getCode(),
                result.getMessage()
            );

        } catch (Exception e) {
            Log.e(TAG, "Failed to handle remote command", e);
        }
    }

    private CommandResult executeCommand(Context context, String type, JSONObject payload) {
        switch (type) {
            case "REBOOT":
                return executeReboot(context);
            case "LOCK":
                return executeLock(context, payload);
            case "FACTORY_RESET":
                return executeFactoryReset(context, payload);
            case "WIPE_DATA":
                return executeWipeData(context, payload);
            case "RING":
                return executeRing(context, payload);
            case "INSTALL_APP":
                return executeInstallApp(context, payload);
            case "UNINSTALL_APP":
                return executeUninstallApp(context, payload);
            default:
                return CommandResult.error("Unknown command type: " + type);
        }
    }

    private CommandResult executeReboot(Context context) {
        try {
            PowerManager pm = (PowerManager) context.getSystemService(Context.POWER_SERVICE);
            pm.reboot("Remote command");
            return CommandResult.success();
        } catch (Exception e) {
            return CommandResult.error(e.getMessage());
        }
    }

    private CommandResult executeLock(Context context, JSONObject payload) {
        try {
            DevicePolicyManager dpm = (DevicePolicyManager)
                context.getSystemService(Context.DEVICE_POLICY_SERVICE);
            ComponentName admin = new ComponentName(context, AdminReceiver.class);

            String message = payload.optString("message", "Device locked by administrator");
            int duration = payload.optInt("duration", 0);

            // 立即锁定
            dpm.lockNow();

            // 设置锁定信息（如果支持）
            if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {
                dpm.setOrganizationName(admin, "MDM Administrator");
                dpm.setShortSupportMessage(admin, message);
            }

            return CommandResult.success();
        } catch (Exception e) {
            return CommandResult.error(e.getMessage());
        }
    }

    private CommandResult executeFactoryReset(Context context, JSONObject payload) {
        try {
            DevicePolicyManager dpm = (DevicePolicyManager)
                context.getSystemService(Context.DEVICE_POLICY_SERVICE);

            boolean preserveES = payload.optBoolean("preserveExternalStorage", false);

            if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.P) {
                // Android 9+ 支持保留外部存储选项
                dpm.wipeData(preserveES ? DevicePolicyManager.WIPE_EXTERNAL_STORAGE : 0);
            } else {
                dpm.wipeData(0);
            }

            return CommandResult.success();
        } catch (Exception e) {
            return CommandResult.error(e.getMessage());
        }
    }

    private CommandResult executeRing(Context context, JSONObject payload) {
        try {
            int duration = payload.optInt("duration", 30);

            // 获取默认铃声
            Ringtone ringtone = RingtoneManager.getRingtone(
                context,
                RingtoneManager.getDefaultUri(RingtoneManager.TYPE_RINGTONE)
            );

            if (ringtone != null) {
                ringtone.play();

                // 设置定时停止
                new Timer().schedule(new TimerTask() {
                    @Override
                    public void run() {
                        if (ringtone.isPlaying()) {
                            ringtone.stop();
                        }
                    }
                }, duration * 1000);

                return CommandResult.success("Ringing for " + duration + " seconds");
            }

            return CommandResult.error("Cannot play ringtone");
        } catch (Exception e) {
            return CommandResult.error(e.getMessage());
        }
    }

    private CommandResult executeInstallApp(Context context, JSONObject payload) {
        try {
            String downloadUrl = payload.getString("downloadUrl");
            String packageName = payload.getString("packageName");

            // 下载并安装APK
            AppInstaller installer = new AppInstaller(context);
            installer.downloadAndInstall(downloadUrl, packageName);

            return CommandResult.success("Installation started");
        } catch (Exception e) {
            return CommandResult.error(e.getMessage());
        }
    }

    private void reportCommandStatus(String commandId, String status,
                                     Integer code, String message) {
        JSONObject statusUpdate = new JSONObject();
        try {
            statusUpdate.put("commandId", commandId);
            statusUpdate.put("status", status);
            statusUpdate.put("code", code);
            statusUpdate.put("message", message);
            statusUpdate.put("timestamp", System.currentTimeMillis());

            ServerConnection.sendCommandStatus(statusUpdate);
        } catch (JSONException e) {
            Log.e(TAG, "Failed to report status", e);
        }
    }
}
```

## 7. 安全考虑

1. **权限验证**: 敏感命令（如擦除数据、恢复出厂设置）需要超级管理员权限
2. **命令签名**: 关键命令使用数字签名验证，防止篡改
3. **超时机制**: 命令有过期时间，防止旧命令被恶意重放
4. **审计日志**: 所有命令操作记录完整的审计日志
5. **设备验证**: 命令只能发送到已注册和授权的设备
6. **确认机制**: 高危操作需要二次确认
