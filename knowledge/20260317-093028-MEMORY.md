## 登录功能优化

2026-03-22 对登录页面进行了功能增强和问题修复：

### 记住密码功能

| 项目 | 说明 |
|------|------|
| 功能描述 | 勾选"记住我"后，下次登录时自动填充用户名 |
| 实现方式 | 使用 localStorage 保存用户名 |
| 安全考虑 | 仅保存用户名，不保存密码 |

**实现文件：**
- `hmdm-server/server/src/main/webapp/app/components/main/controller/login.controller.js`

**核心代码：**
```javascript
// 页面加载时读取保存的用户名
var savedCredentials = localStorage.getItem('hmdm-remembered');
if (savedCredentials) {
    var creds = JSON.parse(savedCredentials);
    $scope.login.username = creds.username || '';
    $scope.login.remember = !!creds.username;
}

// 登录成功时保存凭据
if ($scope.login.remember && $scope.login.username) {
    localStorage.setItem('hmdm-remembered', JSON.stringify({
        username: $scope.login.username
    }));
}
```

| 项目 | 说明 |

## 代码审核与修复

2026-03-18 完成了对新增REST API的代码审核，并修复了8个问题：

### 审核发现的问题

| # | 问题类型 | 问题描述 | 严重程度 |
|---|---------|----------|----------|
| 1 | 性能问题 | 批量操作使用循环单条插入 | 高 |
| 2 | 安全问题 | 异常信息泄露内部错误细节 | 高 |
| 3 | 安全问题 | 危险操作(factory_reset)缺少额外验证 | 中 |
| 4 | 性能问题 | 缺少 @Singleton 注解 | 低 |
| 5 | 可观测性 | 缺少日志记录 | 低 |
| 6 | 功能缺失 | 删除联系人功能未实现 | 中 |
| 7 | 代码规范 | License注释格式不统一 | 低 |
| 8 | 输入验证 | 部分参数缺少验证 | 中 |

### 修复内容

1. **@Singleton注解** - 所有5个REST资源类添加该注解提升性能
2. **日志记录** - 添加SLF4J记录关键操作（设备上线、命令发送、错误日志）
3. **异常处理** - 移除异常堆栈信息泄露，使用通用错误消息
4. **危险操作验证** - factory_reset操作添加日志警告
5. **参数验证** - 文件名类型验证、批量大小限制、输入清理
6. **批量操作限制** - 网络日志上传限制单次最多1000条
7. **删除功能** - 实现 deleteContactsByDevice 方法
8. **代码规范** - 统一使用英文Apache License头
