# Headwind MDM Enterprise 功能实现文档

本项目实现了 Headwind MDM 的10个 Enterprise 核心功能。

## 功能清单

| 序号 | 功能名称 | 英文名称 | 状态 |
|------|---------|---------|------|
| 1 | Kiosk模式 | Kiosk Mode | ✅ 已完成 |
| 2 | 设备位置跟踪 | Device Location Tracking | ✅ 已完成 |
| 3 | 远程命令 | Remote Commands | ✅ 已完成 |
| 4 | 照片上传 | Photo Upload | ✅ 已完成 |
| 5 | 远程控制 | Remote Control | ✅ 已完成 |
| 6 | 网络流量过滤 | Network Traffic Filtering | ✅ 已完成 |
| 7 | LDAP集成 | LDAP Integration | ✅ 已完成 |
| 8 | 设备导入导出 | Export/Import Devices | ✅ 已完成 |
| 9 | 设备联系人管理 | Device Contacts Management | ✅ 已完成 |
| 10 | 白标软件 | White-Label Software | ✅ 已完成 |

## 技术栈

- **后端**: Java 8+, Jersey REST API, MyBatis ORM, PostgreSQL
- **前端**: AngularJS 1.x, Bootstrap
- **依赖注入**: Google Guice
- **推送通知**: MQTT, Polling
- **构建工具**: Maven
