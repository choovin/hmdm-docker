# Feature 5: 远程控制 (Remote Control)

## 1. 功能概述

远程控制功能允许管理员实时查看和控制设备屏幕，基于WebRTC技术实现低延迟、高质量的屏幕共享和远程操作。

### 核心功能
- 实时屏幕共享（查看/控制）
- WebRTC点对点连接
- 远程触摸操作
- 远程按键操作
- 会话录制
- 多管理员同时观看
- 连接质量监控

## 3. 数据库设计

### 3.1 远程控制会话表 (remote_control_sessions)

```sql
CREATE TABLE remote_control_sessions (
    id SERIAL PRIMARY KEY,
    customer_id INTEGER NOT NULL,
    device_id INTEGER REFERENCES devices(id) ON DELETE CASCADE,

    -- 会话信息
    session_token VARCHAR(64) UNIQUE NOT NULL,
    session_type VARCHAR(20) DEFAULT 'VIEW', -- VIEW, CONTROL

    -- 状态: PENDING, CONNECTING, CONNECTED, DISCONNECTED, ERROR
    status VARCHAR(20) DEFAULT 'PENDING',

    -- 连接信息
    webrtc_offer TEXT,
    webrtc_answer TEXT,
    ice_candidates TEXT, -- JSON数组

    -- 时间戳
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    started_at TIMESTAMP,
    ended_at TIMESTAMP,

    -- 质量指标
    connection_quality VARCHAR(20), -- EXCELLENT, GOOD, FAIR, POOR
    latency_ms INTEGER,
    frame_rate DECIMAL(4, 2),

    -- 录制
    is_recorded BOOLEAN DEFAULT FALSE,
    recording_path VARCHAR(500),

    -- 创建者
    created_by INTEGER REFERENCES users(id),

    -- 错误信息
    error_message TEXT,

    INDEX idx_device_status (device_id, status),
    INDEX idx_token (session_token),
    INDEX idx_created_at (created_at)
);
```

### 6.2 RemoteControlServiceDAO

```java
@Singleton
public class RemoteControlServiceDAO {
    private final RemoteControlMapper remoteControlMapper;
    private final PushService pushService;

    @Inject
    public RemoteControlServiceDAO(
        RemoteControlMapper remoteControlMapper,
        PushService pushService
    ) {
        this.remoteControlMapper = remoteControlMapper;
        this.pushService = pushService;
    }

    @Transactional
    public RemoteControlSession createSession(RemoteControlSession session) {
        User currentUser = SecurityContext.get().getCurrentUser().get();

        session.setCustomerId(currentUser.getCustomerId());
        session.setCreatedBy(currentUser.getId());
        session.setSessionToken(generateSessionToken());
        session.setStatus("PENDING");

        remoteControlMapper.insertSession(session);

        // 发送远程控制请求
        sendRemoteControlRequest(session);

        return session;
    }

    private void sendRemoteControlRequest(RemoteControlSession session) {
        ObjectNode payload = JsonMapper.createObjectNode();
        payload.put("sessionId", session.getId());
        payload.put("sessionToken", session.getSessionToken());
        payload.put("sessionType", session.getSessionType());
        payload.put("requestedBy", session.getCreatedBy());

        boolean sent = pushService.sendMessage(
            session.getDeviceId(),
            PushMessage.TYPE_REMOTE_CONTROL_REQUEST,
            payload.toString()
        );

        if (!sent) {
            remoteControlMapper.updateSessionStatus(
                session.getId(), "ERROR", null, "Failed to send push notification"
            );
        }
    }

    public RemoteControlSession getSessionByToken(String token) {
        return remoteControlMapper.getSessionByToken(token);
    }

    public List<RemoteControlSession> getActiveSessionsByDevice(Integer deviceId) {
        return remoteControlMapper.getActiveSessionsByDevice(deviceId);
    }

    @Transactional
    public void updateSessionStatus(Integer sessionId, String status, String errorMessage) {
        remoteControlMapper.updateSessionStatus(sessionId, status, errorMessage);
    }

    @Transactional
    public void updateWebrtcAnswer(Integer sessionId, String answer) {
        remoteControlMapper.updateWebrtcAnswer(sessionId, answer);
    }

    @Transactional
    public void addIceCandidate(Integer sessionId, String candidate) {
        remoteControlMapper.addIceCandidate(sessionId, candidate);
    }

    @Transactional
    public void closeSession(Integer sessionId) {
        RemoteControlSession session = remoteControlMapper.getSessionById(sessionId);
        if (session != null) {
            // 发送停止指令
            ObjectNode payload = JsonMapper.createObjectNode();
            payload.put("sessionId", sessionId);
            payload.put("action", "STOP");

            pushService.sendMessage(
                session.getDeviceId(),
                PushMessage.TYPE_REMOTE_CONTROL_STOP,
                payload.toString()
            );

            remoteControlMapper.updateSessionStatus(sessionId, "DISCONNECTED", null, null);
        }
    }

    private String generateSessionToken() {
        return UUID.randomUUID().toString().replace("-", "") +
               System.currentTimeMillis();
    }
}
```

### 6.3 Android Agent端WebRTC服务

```java
// RemoteControlService.java
public class RemoteControlService extends Service {
    private PeerConnectionFactory peerConnectionFactory;
    private PeerConnection peerConnection;
    private EglBase eglBase;
    private ScreenCapturer screenCapturer;
    private VideoSource videoSource;
    private SurfaceTextureHelper surfaceTextureHelper;

    private static final String STUN_SERVER = "stun:stun.l.google.com:19302";
    private static final List<String> TURN_SERVERS = Arrays.asList(
        "turn:turn.example.com:3478"
    );

    @Override
    public void onCreate() {
        super.onCreate();
        initializePeerConnectionFactory();
    }

    private void initializePeerConnectionFactory() {
        eglBase = EglBase.create();

        PeerConnectionFactory.InitializationOptions options =
            PeerConnectionFactory.InitializationOptions.builder(this)
                .setEnableInternalTracer(true)
                .createInitializationOptions();

        PeerConnectionFactory.initialize(options);

        PeerConnectionFactory.Options factoryOptions = new PeerConnectionFactory.Options();
        factoryOptions.networkIgnoreMask = 0;

        peerConnectionFactory = PeerConnectionFactory.builder()
            .setVideoEncoderFactory(new DefaultVideoEncoderFactory(eglBase.getEglBaseContext(), true, true))
            .setVideoDecoderFactory(new DefaultVideoDecoderFactory(eglBase.getEglBaseContext()))
            .setOptions(factoryOptions)
            .createPeerConnectionFactory();
    }

    public void startSession(String sessionToken, String sessionType) {
        // 创建PeerConnection
        createPeerConnection();

        // 创建屏幕捕获
        createScreenCapturer();

        // 创建VideoTrack
        videoSource = peerConnectionFactory.createVideoSource(screenCapturer.isScreencast());
        screenCapturer.initialize(surfaceTextureHelper, getApplicationContext(), videoSource.getCapturerObserver());
        screenCapturer.startCapture(1080, 2280, 30);

        VideoTrack videoTrack = peerConnectionFactory.createVideoTrack("screen", videoSource);

        // 添加Track到PeerConnection
        MediaStream mediaStream = peerConnectionFactory.createLocalMediaStream("stream");
        mediaStream.addTrack(videoTrack);
        peerConnection.addStream(mediaStream);

        // 创建Offer
        createOffer();
    }

    private void createPeerConnection() {
        List<PeerConnection.IceServer> iceServers = new ArrayList<>();

        // 添加STUN服务器
        iceServers.add(PeerConnection.IceServer.builder(STUN_SERVER).createIceServer());

        // 添加TURN服务器
        for (String turnUrl : TURN_SERVERS) {
            iceServers.add(PeerConnection.IceServer.builder(turnUrl)
                .setUsername("user")
                .setPassword("pass")
                .createIceServer());
        }

        PeerConnection.RTCConfiguration rtcConfig = new PeerConnection.RTCConfiguration(iceServers);
        rtcConfig.tcpCandidatePolicy = PeerConnection.TcpCandidatePolicy.DISABLED;
        rtcConfig.bundlePolicy = PeerConnection.BundlePolicy.MAXBUNDLE;
        rtcConfig.rtcpMuxPolicy = PeerConnection.RtcpMuxPolicy.REQUIRE;
        rtcConfig.continualGatheringPolicy = PeerConnection.ContinualGatheringPolicy.GATHER_CONTINUALLY;
        rtcConfig.keyType = PeerConnection.KeyType.ECDSA;

        peerConnection = peerConnectionFactory.createPeerConnection(rtcConfig, peerConnectionObserver);
    }

    private void createScreenCapturer() {
        MediaProjectionManager projectionManager =
            (MediaProjectionManager) getSystemService(Context.MEDIA_PROJECTION_SERVICE);

        // 需要用户授权MediaProjection
        Intent captureIntent = projectionManager.createScreenCaptureIntent();
        // ... 启动授权流程

        screenCapturer = new ScreenCapturerAndroid(captureIntent, mediaProjectionCallback);
        surfaceTextureHelper = SurfaceTextureHelper.create("ScreenCaptureThread", eglBase.getEglBaseContext());
    }

    private void createOffer() {
        MediaConstraints constraints = new MediaConstraints();
        constraints.mandatory.add(new MediaConstraints.KeyValuePair("OfferToReceiveVideo", "true"));

        peerConnection.createOffer(new SdpObserver() {
            @Override
            public void onCreateSuccess(SessionDescription sessionDescription) {
                peerConnection.setLocalDescription(this, sessionDescription);

                // 发送Offer到服务器
                sendOfferToServer(sessionDescription);
            }

            @Override
            public void onCreateFailure(String s) {
                Log.e(TAG, "Failed to create offer: " + s);
            }
        }, constraints);
    }

    public void handleRemoteAnswer(SessionDescription answer) {
        peerConnection.setRemoteDescription(new SdpObserver() {
            @Override

                break;
            case "KEY":

            case "SWIPE":

    private void handleTouchEvent(JSONObject eventData) {
        String action = eventData.optString("action");
        float x = (float) eventData.optDouble("x", 0);
        float y = (float) eventData.optDouble("y", 0);

        // 使用InputManager注入触摸事件

        long now = SystemClock.uptimeMillis();

        MotionEvent event = MotionEvent.obtain(now, now,
            action.equals("DOWN") ? MotionEvent.ACTION_DOWN : MotionEvent.ACTION_UP,
            x, y, 0);

        injectInputEvent(event);

            injectMethod.invoke(im, event, 0);
        } catch (Exception e) {
            Log.e(TAG, "Failed to inject input event", e);
        }
    }
```

### 6.4 远程控制命令接收器

```java
public class RemoteControlReceiver extends BroadcastReceiver {
    @Override
    public void onReceive(Context context, Intent intent) {
        String messageType = intent.getStringExtra("messageType");

        if (PushMessage.TYPE_REMOTE_CONTROL_REQUEST.equals(messageType)) {
            handleRemoteControlRequest(context, intent);
        } else if (PushMessage.TYPE_REMOTE_CONTROL_STOP.equals(messageType)) {
            handleRemoteControlStop(context, intent);
        }
    }

    private void handleRemoteControlRequest(Context context, Intent intent) {
        String payload = intent.getStringExtra("payload");
        try {
            JSONObject request = new JSONObject(payload);
            int sessionId = request.getInt("sessionId");
            String sessionToken = request.getString("sessionToken");
            String sessionType = request.optString("sessionType", "VIEW");

            // 检查权限（需要设备Owner或特殊权限）
            if (!hasRemoteControlPermission()) {
                // 拒绝请求
                rejectSession(sessionId, "No permission");
                return;
            }

            // 启动远程控制服务
            Intent serviceIntent = new Intent(context, RemoteControlService.class);
            serviceIntent.putExtra("sessionId", sessionId);
            serviceIntent.putExtra("sessionToken", sessionToken);
            serviceIntent.putExtra("sessionType", sessionType);

            if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {
                context.startForegroundService(serviceIntent);
            } else {
                context.startService(serviceIntent);
            }

        } catch (JSONException e) {
            Log.e(TAG, "Failed to parse request", e);
        }
    }

    private void handleRemoteControlStop(Context context, Intent intent) {
        String payload = intent.getStringExtra("payload");
        try {
            JSONObject request = new JSONObject(payload);
            int sessionId = request.getInt("sessionId");

            // 发送停止指令到服务
            Intent stopIntent = new Intent(context, RemoteControlService.class);
            stopIntent.setAction("STOP_SESSION");
            stopIntent.putExtra("sessionId", sessionId);
            context.startService(stopIntent);

        } catch (JSONException e) {
            Log.e(TAG, "Failed to parse stop request", e);
        }
    }
}
```

## 7. 安全考虑

1. **用户确认**: 远程控制开始前需要设备端用户确认
2. **权限控制**: 只有设备Owner才能进行远程控制
3. **会话加密**: WebRTC连接使用DTLS-SRTP加密
4. **会话超时**: 会话自动超时断开，防止长期连接
5. **审计日志**: 记录所有远程控制会话
6. **单会话限制**: 同一设备同一时间只允许一个控制会话
7. **敏感界面检测**: 检测到敏感界面（如密码输入）时暂停控制
