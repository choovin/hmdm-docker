# Feature 1: Kiosk模式 (Kiosk Mode)

## 1. 功能概述

Kiosk模式允许管理员将设备锁定在单一应用或特定应用集合中，限制用户访问设备其他功能。适用于数字标牌、POS系统、自助服务终端等场景。

### 核心功能
- 设置Kiosk模式配置（单应用/多应用）
- 配置禁止启动的应用列表
- 设置系统UI可见性（状态栏、导航栏等）
- 阻止应用卸载
- 阻止应用安装
- Kiosk模式事件日志记录

## 3. 数据库设计

### 3.1 Kiosk配置表 (kiosk_configurations)

```sql
CREATE TABLE kiosk_configurations (
    id SERIAL PRIMARY KEY,
    configuration_id INTEGER REFERENCES configurations(id) ON DELETE CASCADE,
    customer_id INTEGER NOT NULL,

    -- Kiosk模式设置
    kiosk_mode VARCHAR(20) DEFAULT 'DISABLED', -- DISABLED, SINGLE_APP, MULTI_APP
    launcher_app_pkg VARCHAR(255), -- Kiosk启动器应用包名

    -- 系统UI设置
    hide_status_bar BOOLEAN DEFAULT FALSE,
    hide_navigation_bar BOOLEAN DEFAULT FALSE,
    allow_settings BOOLEAN DEFAULT FALSE,
    allow_recent_apps BOOLEAN DEFAULT FALSE,
    allow_power_off BOOLEAN DEFAULT FALSE,

    -- 安全设置
    block_uninstall BOOLEAN DEFAULT FALSE,
    block_install BOOLEAN DEFAULT FALSE,
    block_camera BOOLEAN DEFAULT FALSE,
    block_screenshot BOOLEAN DEFAULT FALSE,

    -- 其他设置
    allowed_apps TEXT, -- JSON数组: ["pkg1", "pkg2"]
    custom_wallpaper VARCHAR(255),
    custom_logo VARCHAR(255),

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    UNIQUE(configuration_id)
);
```

### 3.2 禁止应用表 (kiosk_prohibited_apps)

```sql
CREATE TABLE kiosk_prohibited_apps (
    id SERIAL PRIMARY KEY,
    customer_id INTEGER NOT NULL,
    configuration_id INTEGER REFERENCES configurations(id) ON DELETE CASCADE,
    application_id INTEGER REFERENCES applications(id) ON DELETE CASCADE,
    package_name VARCHAR(255) NOT NULL,
    app_name VARCHAR(255),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 3.3 Kiosk事件日志表 (kiosk_events)

```sql
CREATE TABLE kiosk_events (
    id SERIAL PRIMARY KEY,
    device_id INTEGER REFERENCES devices(id) ON DELETE CASCADE,
    customer_id INTEGER NOT NULL,
    event_type VARCHAR(50) NOT NULL, -- APP_BLOCKED, EXIT_ATTEMPT, SETTINGS_BLOCKED, etc.
    package_name VARCHAR(255),
    details TEXT,
    event_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 6.2 KioskConfigurationDAO

```java
@Singleton
public class KioskConfigurationDAO {
    private final KioskConfigurationMapper mapper;
    private final PushService pushService;

    @Inject
    public KioskConfigurationDAO(KioskConfigurationMapper mapper, PushService pushService) {
        this.mapper = mapper;
        this.pushService = pushService;
    }

    public KioskConfiguration getKioskConfig(Integer configurationId) {
        return mapper.getKioskConfigByConfigurationId(
            configurationId,
            SecurityContext.get().getCurrentUser().get().getCustomerId()
        );
    }

    @Transactional
    public KioskConfiguration saveKioskConfig(KioskConfiguration config) {
        config.setCustomerId(SecurityContext.get().getCurrentUser().get().getCustomerId());
        if (config.getId() == null) {
            mapper.insertKioskConfig(config);
        } else {
            mapper.updateKioskConfig(config);
        }
        // 通知所有使用该配置的设备更新
        notifyDevicesOnKioskConfigUpdate(config.getConfigurationId());
        return config;
    }

    private void notifyDevicesOnKioskConfigUpdate(Integer configurationId) {
        // 获取使用该配置的所有设备并发送推送通知
        pushService.notifyDevicesOnKioskConfigUpdate(configurationId);
    }
}
```

### 6.3 Android Agent端处理

```java
// KioskModeReceiver.java
public class KioskModeReceiver extends BroadcastReceiver {
    @Override
    public void onReceive(Context context, Intent intent) {
        String messageType = intent.getStringExtra("messageType");
        if (PushMessage.TYPE_KIOSK_MODE_CONFIG.equals(messageType)) {
            // 应用新的Kiosk配置
            KioskConfig config = parseConfig(intent.getStringExtra("payload"));
            applyKioskMode(context, config);
        }
    }

    private void applyKioskMode(Context context, KioskConfig config) {
        DevicePolicyManager dpm = (DevicePolicyManager)
            context.getSystemService(Context.DEVICE_POLICY_SERVICE);
        ComponentName admin = new ComponentName(context, AdminReceiver.class);

        if ("SINGLE_APP".equals(config.getKioskMode())) {
            // 单应用模式 - 设置设备 owner 锁定任务
            dpm.setLockTaskPackages(admin, new String[]{config.getLauncherAppPkg()});
        } else if ("MULTI_APP".equals(config.getKioskMode())) {
            // 多应用模式
            List<String> allowedApps = parseAllowedApps(config.getAllowedApps());
            dpm.setLockTaskPackages(admin, allowedApps.toArray(new String[0]));
        }

        // 应用系统UI设置
        if (config.getHideStatusBar()) {
            hideStatusBar(context);
        }
        if (config.getHideNavigationBar()) {
            hideNavigationBar(context);
        }

        // 启动Kiosk应用
        Intent launcherIntent = context.getPackageManager()
            .getLaunchIntentForPackage(config.getLauncherAppPkg());
        if (launcherIntent != null) {
            launcherIntent.addFlags(Intent.FLAG_ACTIVITY_NEW_TASK);
            context.startActivity(launcherIntent);
        }
    }
}
```

## 7. 安全考虑

1. **权限控制**: 只有设备管理员才能启用Kiosk模式
2. **退出保护**: 需要特定密码或管理员指令才能退出Kiosk模式
3. **事件审计**: 所有Kiosk相关事件都记录到数据库，便于审计
4. **配置验证**: 确保Kiosk配置在应用到设备前经过验证
