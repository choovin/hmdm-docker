# Feature 2: 设备位置跟踪 (Device Location Tracking)

## 1. 功能概述

设备位置跟踪功能允许管理员实时查看和管理设备的地理位置信息，支持地理围栏(Geofencing)功能，当设备进入或离开指定区域时触发警报。

### 核心功能
- 实时获取设备GPS位置
- 历史位置轨迹查询
- 地理围栏(Geofence)管理
- 进出围栏事件通知
- 位置数据可视化（地图集成）
- 位置更新频率配置

## 3. 数据库设计

### 3.1 设备位置表 (device_locations)

```sql
CREATE TABLE device_locations (
    id SERIAL PRIMARY KEY,
    device_id INTEGER REFERENCES devices(id) ON DELETE CASCADE,
    customer_id INTEGER NOT NULL,

    -- 位置信息
    latitude DECIMAL(10, 8) NOT NULL,
    longitude DECIMAL(11, 8) NOT NULL,
    altitude DECIMAL(10, 2),
    accuracy DECIMAL(8, 2),
    speed DECIMAL(8, 2),
    bearing DECIMAL(5, 2),

    -- 地址信息（反向地理编码）
    address TEXT,
    city VARCHAR(100),
    country VARCHAR(100),

    -- 时间戳
    location_time TIMESTAMP NOT NULL,
    received_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    -- 索引
    INDEX idx_device_time (device_id, location_time),
    INDEX idx_location (latitude, longitude)
);
```

### 3.2 地理围栏表 (geofences)

```sql
CREATE TABLE geofences (
    id SERIAL PRIMARY KEY,
    customer_id INTEGER NOT NULL,
    configuration_id INTEGER REFERENCES configurations(id) ON DELETE CASCADE,

    -- 围栏基本信息
    name VARCHAR(255) NOT NULL,
    description TEXT,

    -- 围栏类型: CIRCLE, POLYGON
    fence_type VARCHAR(20) DEFAULT 'CIRCLE',

    -- 圆形围栏参数
    center_latitude DECIMAL(10, 8),
    center_longitude DECIMAL(11, 8),
    radius_meters INTEGER,

    -- 多边形围栏参数 (JSON格式存储顶点坐标)
    polygon_points TEXT, -- [{"lat": x, "lng": y}, ...]

    -- 触发条件
    trigger_on_enter BOOLEAN DEFAULT TRUE,
    trigger_on_exit BOOLEAN DEFAULT TRUE,

    -- 关联动作
    send_notification BOOLEAN DEFAULT TRUE,
    notification_emails TEXT, -- JSON数组
    auto_lock_device BOOLEAN DEFAULT FALSE,
    auto_wipe_data BOOLEAN DEFAULT FALSE,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 3.3 地理围栏事件表 (geofence_events)

```sql
CREATE TABLE geofence_events (
    id SERIAL PRIMARY KEY,
    geofence_id INTEGER REFERENCES geofences(id) ON DELETE CASCADE,
    device_id INTEGER REFERENCES devices(id) ON DELETE CASCADE,
    customer_id INTEGER NOT NULL,

    -- 事件类型: ENTER, EXIT
    event_type VARCHAR(20) NOT NULL,

    -- 事件发生时位置
    latitude DECIMAL(10, 8) NOT NULL,
    longitude DECIMAL(11, 8) NOT NULL,

    -- 时间戳
    event_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    -- 处理状态
    is_processed BOOLEAN DEFAULT FALSE,
    processed_at TIMESTAMP,

    INDEX idx_device_event (device_id, event_time),
    INDEX idx_geofence (geofence_id, event_time)
);
```

### 6.3 LocationServiceDAO

```java
@Singleton
public class LocationServiceDAO {
    private final DeviceLocationMapper locationMapper;
    private final GeofenceMapper geofenceMapper;
    private final PushService pushService;

    @Inject
    public LocationServiceDAO(
        DeviceLocationMapper locationMapper,
        GeofenceMapper geofenceMapper,
        PushService pushService
    ) {
        this.locationMapper = locationMapper;
        this.geofenceMapper = geofenceMapper;
        this.pushService = pushService;
    }

    public DeviceLocation getLatestLocation(Integer deviceId) {
        return locationMapper.getLatestLocation(
            deviceId,
            SecurityContext.get().getCurrentUser().get().getCustomerId()
        );
    }

    public List<DeviceLocation> getLocationHistory(
        Integer deviceId, Long fromTime, Long toTime, Integer limit
    ) {
        return locationMapper.getLocationHistory(
            deviceId,
            SecurityContext.get().getCurrentUser().get().getCustomerId(),
            new Date(fromTime),
            new Date(toTime),
            limit
        );
    }

    @Transactional
    public DeviceLocation saveLocation(DeviceLocation location) {
        locationMapper.insertLocation(location);

        // 检查地理围栏触发
        checkGeofenceTriggers(location);

        return location;
    }

    private void checkGeofenceTriggers(DeviceLocation location) {
        List<Geofence> geofences = geofenceMapper.getActiveGeofencesForDevice(
            location.getDeviceId()
        );

        for (Geofence geofence : geofences) {
            boolean isInside = isPointInGeofence(
                location.getLatitude(),
                location.getLongitude(),
                geofence
            );

            GeofenceEvent lastEvent = geofenceMapper.getLastEvent(
                geofence.getId(),
                location.getDeviceId()
            );

            boolean wasInside = lastEvent != null &&
                "ENTER".equals(lastEvent.getEventType());

            // 触发进入事件
            if (isInside && !wasInside && geofence.getTriggerOnEnter()) {
                triggerGeofenceEvent(geofence, location, "ENTER");
            }

            // 触发离开事件
            if (!isInside && wasInside && geofence.getTriggerOnExit()) {
                triggerGeofenceEvent(geofence, location, "EXIT");
            }
        }
    }

    private boolean isPointInGeofence(BigDecimal lat, BigDecimal lng, Geofence fence) {
        if ("CIRCLE".equals(fence.getFenceType())) {
            double distance = calculateDistance(
                lat.doubleValue(), lng.doubleValue(),
                fence.getCenterLatitude().doubleValue(),
                fence.getCenterLongitude().doubleValue()
            );
            return distance <= fence.getRadiusMeters();
        } else {
            // 多边形检测逻辑
            return isPointInPolygon(lat, lng, fence.getPolygonPoints());
        }
    }

    public void requestLocationUpdate(Integer deviceId) {
        pushService.sendMessage(deviceId, PushMessage.TYPE_LOCATION_REQUEST, null);
    }
}
```

### 6.4 Android Agent端位置服务

```java
// LocationUpdateService.java
public class LocationUpdateService extends Service {
    private FusedLocationProviderClient fusedLocationClient;
    private LocationCallback locationCallback;

    @Override
    public void onCreate() {
        super.onCreate();
        fusedLocationClient = LocationServices.getFusedLocationProviderClient(this);

        locationCallback = new LocationCallback() {
            @Override
            public void onLocationResult(LocationResult result) {
                if (result.getLastLocation() != null) {
                    sendLocationToServer(result.getLastLocation());
                }
            }
        };
    }

    private void startLocationUpdates() {
        LocationRequest request = LocationRequest.create()
            .setPriority(LocationRequest.PRIORITY_BALANCED_POWER_ACCURACY)
            .setInterval(60000) // 60秒更新一次
            .setFastestInterval(30000);

        try {
            fusedLocationClient.requestLocationUpdates(
                request, locationCallback, Looper.getMainLooper()
            );
        } catch (SecurityException e) {
            Log.e(TAG, "Location permission not granted", e);
        }
    }

    private void sendLocationToServer(Location location) {
        DeviceLocation deviceLocation = new DeviceLocation();
        deviceLocation.setDeviceId(getDeviceId());
        deviceLocation.setLatitude(BigDecimal.valueOf(location.getLatitude()));
        deviceLocation.setLongitude(BigDecimal.valueOf(location.getLongitude()));
        deviceLocation.setAltitude(BigDecimal.valueOf(location.getAltitude()));
        deviceLocation.setAccuracy(BigDecimal.valueOf(location.getAccuracy()));
        deviceLocation.setSpeed(BigDecimal.valueOf(location.getSpeed()));
        deviceLocation.setBearing(BigDecimal.valueOf(location.getBearing()));
        deviceLocation.setLocationTime(new Date());

        // 发送到服务器
        ServerConnection.sendLocationUpdate(deviceLocation);
    }

    // 处理服务器位置请求
    public void onLocationRequestReceived() {
        try {
            fusedLocationClient.getLastLocation()
                .addOnSuccessListener(location -> {
                    if (location != null) {
                        sendLocationToServer(location);
                    }
                });
        } catch (SecurityException e) {
            Log.e(TAG, "Cannot get location", e);
        }
    }
}
```

### 6.5 地理围栏检测服务

```java
// GeofenceBroadcastReceiver.java
public class GeofenceBroadcastReceiver extends BroadcastReceiver {
    @Override
    public void onReceive(Context context, Intent intent) {
        GeofencingEvent event = GeofencingEvent.fromIntent(intent);

        if (event.hasError()) {
            Log.e(TAG, "Geofence error: " + event.getErrorCode());
            return;
        }

        int transition = event.getGeofenceTransition();
        List<Geofence> triggeredGeofences = event.getTriggeringGeofences();
        Location triggerLocation = event.getTriggeringLocation();

        for (Geofence geofence : triggeredGeofences) {
            String eventType = transition == Geofence.GEOFENCE_TRANSITION_ENTER
                ? "ENTER" : "EXIT";

            // 上报围栏事件
            reportGeofenceEvent(geofence.getRequestId(), eventType, triggerLocation);

            // 执行本地动作
            if (transition == Geofence.GEOFENCE_TRANSITION_EXIT) {
                // 检查是否需要自动锁定
                if (shouldAutoLock(geofence.getRequestId())) {
                    lockDevice(context);
                }
            }
        }
    }

    private void reportGeofenceEvent(String geofenceId, String eventType, Location location) {
        GeofenceEvent event = new GeofenceEvent();
        event.setGeofenceId(Integer.parseInt(geofenceId));
        event.setEventType(eventType);
        event.setLatitude(BigDecimal.valueOf(location.getLatitude()));
        event.setLongitude(BigDecimal.valueOf(location.getLongitude()));
        event.setEventTime(new Date());

        ServerConnection.sendGeofenceEvent(event);
    }
}
```

## 7. 安全考虑

1. **位置隐私**: 位置数据属于敏感信息，仅授权用户可访问
2. **数据传输加密**: 所有位置数据通过HTTPS传输
3. **数据保留策略**: 支持配置位置数据保留期限，自动清理过期数据
4. **权限控制**: 细粒度的位置查看权限管理
5. **地理围栏安全**: 围栏事件通知使用可靠的消息队列确保不丢失
