# Feature 8: 设备导入导出 (Export/Import Devices)

## 1. 功能概述

设备导入导出功能允许管理员批量导入设备信息，或导出设备列表，支持CSV和Excel格式，便于设备管理和数据迁移。

### 核心功能
- CSV/Excel批量导入设备
- 设备数据导出
- 导入模板下载
- 导入数据验证
- 导入进度跟踪
- 导出字段选择
- 历史记录管理

## 3. 数据库设计

### 3.1 设备导入记录表 (device_imports)

```sql
CREATE TABLE device_imports (
    id SERIAL PRIMARY KEY,
    customer_id INTEGER NOT NULL,

    -- 文件信息
    file_name VARCHAR(255) NOT NULL,
    file_size BIGINT,

    -- 导入状态: PROCESSING, COMPLETED, FAILED
    status VARCHAR(20) DEFAULT 'PROCESSING',

    -- 统计
    total_count INTEGER DEFAULT 0,
    success_count INTEGER DEFAULT 0,
    failed_count INTEGER DEFAULT 0,
    skipped_count INTEGER DEFAULT 0,

    -- 错误详情
    error_details TEXT, -- JSON格式

    -- 时间戳
    started_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    completed_at TIMESTAMP,

    -- 发起人
    created_by INTEGER REFERENCES users(id),

    INDEX idx_customer_status (customer_id, status),
    INDEX idx_started_at (started_at)
);
```

## 6. 关键代码示例

### 6.1 设备导入服务

```java
@Singleton
public class DeviceImportService {
    private final DeviceDAO deviceDAO;
    private final GroupDAO groupDAO;
    private final ConfigurationDAO configurationDAO;
    private final DeviceImportMapper importMapper;

    @Transactional
    public DeviceImportResult importDevices(InputStream fileStream,
                                            String fileName,
                                            ImportOptions options) {
        DeviceImport importRecord = new DeviceImport();
        importRecord.setFileName(fileName);
        importRecord.setStatus("PROCESSING");
        importRecord.setStartedAt(System.currentTimeMillis());
        importRecord.setCreatedBy(getCurrentUserId());

        importMapper.insert(importRecord);

        DeviceImportResult result = new DeviceImportResult();
        List<ImportError> errors = new ArrayList<>();

        try {
            List<DeviceImportRow> rows = parseImportFile(fileStream, fileName);
            importRecord.setTotalCount(rows.size());

            for (int i = 0; i < rows.size(); i++) {
                DeviceImportRow row = rows.get(i);

                try {
                    ImportRowResult rowResult = processImportRow(row, options);

                    switch (rowResult.getAction()) {
                        case CREATED:
                            result.incrementSuccess();
                            break;
                        case UPDATED:
                            result.incrementSuccess();
                            break;
                        case SKIPPED:
                            result.incrementSkipped();
                            break;
                    }

                } catch (ImportException e) {
                    result.incrementFailed();
                    errors.add(new ImportError(i + 1, row.getImei(), e.getMessage()));
                }

                // 更新进度
                if (i % 10 == 0) {
                    updateProgress(importRecord.getId(), i + 1,
                        result.getSuccessCount(), result.getFailedCount());
                }
            }

            importRecord.setStatus("COMPLETED");
            importRecord.setCompletedAt(System.currentTimeMillis());

        } catch (Exception e) {
            importRecord.setStatus("FAILED");
            errors.add(new ImportError(0, "N/A", "Import failed: " + e.getMessage()));
        }

        importRecord.setSuccessCount(result.getSuccessCount());
        importRecord.setFailedCount(result.getFailedCount());
        importRecord.setSkippedCount(result.getSkippedCount());
        importRecord.setErrorDetails(JsonMapper.toJson(errors));

        importMapper.update(importRecord);

        return result;
    }

    private List<DeviceImportRow> parseImportFile(InputStream stream, String fileName)
        throws ImportException {

        String extension = getFileExtension(fileName).toLowerCase();

        switch (extension) {
            case "csv":
                return parseCsv(stream);
            case "xlsx":
            case "xls":
                return parseExcel(stream);
            default:
                throw new ImportException("Unsupported file format: " + extension);
        }
    }

    private List<DeviceImportRow> parseCsv(InputStream stream) throws ImportException {
        List<DeviceImportRow> rows = new ArrayList<>();

        try (BufferedReader reader = new BufferedReader(
            new InputStreamReader(stream, StandardCharsets.UTF_8))) {

            CSVParser parser = new CSVParser(reader, CSVFormat.DEFAULT
                .withFirstRecordAsHeader()
                .withIgnoreHeaderCase()
                .withTrim());

            for (CSVRecord record : parser) {
                DeviceImportRow row = new DeviceImportRow();
                row.setImei(record.get("IMEI"));
                row.setDeviceName(record.get("设备名称"));
                row.setGroupName(record.get("设备组"));
                row.setConfigurationName(record.get("配置"));
                row.setDescription(record.get("描述"));
                row.setCustomFields(record.get("自定义字段"));
                rows.add(row);
            }

        } catch (IOException e) {
            throw new ImportException("Failed to parse CSV: " + e.getMessage());
        }

        return rows;
    }

    private ImportRowResult processImportRow(DeviceImportRow row, ImportOptions options) {
        // 验证IMEI
        if (!isValidImei(row.getImei())) {
            throw new ImportException("Invalid IMEI: " + row.getImei());
        }

        // 查找现有设备
        Device existingDevice = deviceDAO.findByImei(row.getImei());

        if (existingDevice != null) {
            if (options.getSkipExisting()) {
                return ImportRowResult.skipped("Device already exists");
            }

            if (options.getUpdateExisting()) {
                // 更新现有设备
                updateDeviceFromRow(existingDevice, row);
                deviceDAO.update(existingDevice);
                return ImportRowResult.updated();
            }
        }

        // 创建新设备
        Device newDevice = createDeviceFromRow(row);
        deviceDAO.insert(newDevice);

        return ImportRowResult.created();
    }

    private Device createDeviceFromRow(DeviceImportRow row) {
        Device device = new Device();
        device.setImei(row.getImei());
        device.setName(row.getDeviceName());
        device.setDescription(row.getDescription());
        device.setCustomerId(getCurrentCustomerId());

        // 设置设备组
        if (StringUtils.isNotBlank(row.getGroupName())) {
            Group group = groupDAO.findByName(row.getGroupName(), getCurrentCustomerId());
            if (group != null) {
                device.setGroupId(group.getId());
            }
        }

        // 设置配置
        if (StringUtils.isNotBlank(row.getConfigurationName())) {
            Configuration config = configurationDAO.findByName(
                row.getConfigurationName(), getCurrentCustomerId());
            if (config != null) {
                device.setConfigurationId(config.getId());
            }
        }

        // 设置自定义字段
        if (StringUtils.isNotBlank(row.getCustomFields())) {
            device.setCustomProperties(parseCustomFields(row.getCustomFields()));
        }

        return device;
    }
}
```

### 6.2 设备导出服务

```java
@Singleton
public class DeviceExportService {
    private final DeviceDAO deviceDAO;

    public byte[] exportDevices(DeviceExportRequest request) {
        List<Device> devices = fetchDevices(request);
        List<String> fields = request.getFields();

        String format = request.getFormat();

        if ("csv".equalsIgnoreCase(format)) {
            return exportToCsv(devices, fields);
        } else if ("xlsx".equalsIgnoreCase(format)) {
            return exportToExcel(devices, fields);
        }

        throw new IllegalArgumentException("Unsupported format: " + format);
    }

    private byte[] exportToExcel(List<Device> devices, List<String> fields) {
        try (Workbook workbook = new XSSFWorkbook();
             ByteArrayOutputStream out = new ByteArrayOutputStream()) {

            Sheet sheet = workbook.createSheet("Devices");

            // 创建标题行
            Row headerRow = sheet.createRow(0);
            CellStyle headerStyle = createHeaderStyle(workbook);

            for (int i = 0; i < fields.size(); i++) {
                Cell cell = headerRow.createCell(i);
                cell.setCellValue(getFieldLabel(fields.get(i)));
                cell.setCellStyle(headerStyle);
            }

            // 填充数据
            int rowNum = 1;
            for (Device device : devices) {
                Row row = sheet.createRow(rowNum++);
                for (int i = 0; i < fields.size(); i++) {
                    Cell cell = row.createCell(i);
                    setCellValue(cell, device, fields.get(i));
                }
            }

            // 自动调整列宽
            for (int i = 0; i < fields.size(); i++) {
                sheet.autoSizeColumn(i);
            }

            workbook.write(out);
            return out.toByteArray();

        } catch (IOException e) {
            throw new RuntimeException("Failed to export devices", e);
        }
    }

    private void setCellValue(Cell cell, Device device, String field) {
        switch (field) {
            case "imei":
                cell.setCellValue(device.getImei());
                break;
            case "name":
                cell.setCellValue(device.getName());
                break;
            case "description":
                cell.setCellValue(device.getDescription());
                break;
            case "group":
                cell.setCellValue(device.getGroupName());
                break;
            case "configuration":
                cell.setCellValue(device.getConfigurationName());
                break;
            case "lastOnline":
                if (device.getLastOnline() != null) {
                    cell.setCellValue(new Date(device.getLastOnline()));
                    cell.setCellStyle(createDateStyle(cell.getSheet().getWorkbook()));
                }
                break;
            case "status":
                cell.setCellValue(device.getStatus());
                break;
            // ... 更多字段
        }
    }
}
```

## 7. 安全考虑

1. **文件验证**: 验证文件类型和大小，防止上传恶意文件
2. **数据验证**: 严格验证导入数据格式
3. **权限控制**: 只有管理员可以导入导出设备
4. **敏感数据**: 导出的敏感数据（如IMEI）需妥善处理
