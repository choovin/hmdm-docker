# Feature 4: 照片上传 (Photo Upload)

## 1. 功能概述

照片上传功能允许管理员远程请求设备拍摄并上传照片，支持从前后摄像头拍照，用于设备定位、现场取证、资产检查等场景。

### 核心功能
- 远程请求设备拍照
- 前后摄像头选择
- 照片元数据记录（位置、时间戳、设备信息）
- 照片管理和浏览
- 照片搜索和过滤
- 批量下载
- 照片质量/分辨率配置

## 3. 数据库设计

### 3.1 照片表 (photos)

```sql
CREATE TABLE photos (
    id SERIAL PRIMARY KEY,
    device_id INTEGER REFERENCES devices(id) ON DELETE CASCADE,
    customer_id INTEGER NOT NULL,

    -- 照片文件信息
    file_name VARCHAR(255) NOT NULL,
    file_path VARCHAR(500) NOT NULL,
    file_size BIGINT,
    mime_type VARCHAR(50),

    -- 照片元数据
    width INTEGER,
    height INTEGER,
    orientation INTEGER DEFAULT 0,

    -- 拍摄位置
    latitude DECIMAL(10, 8),
    longitude DECIMAL(11, 8),
    address TEXT,

    -- 相机信息
    camera_type VARCHAR(20), -- FRONT, BACK
    flash_used BOOLEAN DEFAULT FALSE,
    focal_length DECIMAL(8, 2),

    -- 时间戳
    taken_at TIMESTAMP,
    uploaded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    -- 请求信息
    request_id INTEGER,
    requested_by INTEGER REFERENCES users(id),

    -- 描述
    description TEXT,
    tags TEXT, -- JSON数组

    -- 删除标记
    is_deleted BOOLEAN DEFAULT FALSE,
    deleted_at TIMESTAMP,

    INDEX idx_device (device_id),
    INDEX idx_uploaded_at (uploaded_at),
    INDEX idx_device_time (device_id, taken_at)
);
```

### 3.2 照片上传请求表 (photo_requests)

```sql
CREATE TABLE photo_requests (
    id SERIAL PRIMARY KEY,
    device_id INTEGER REFERENCES devices(id) ON DELETE CASCADE,
    customer_id INTEGER NOT NULL,

    -- 请求信息
    camera_type VARCHAR(20) DEFAULT 'BACK', -- FRONT, BACK, BOTH
    resolution VARCHAR(20) DEFAULT 'MEDIUM', -- LOW, MEDIUM, HIGH
    auto_upload BOOLEAN DEFAULT TRUE,
    include_location BOOLEAN DEFAULT TRUE,

    -- 状态: PENDING, SENT, CAPTURING, UPLOADING, COMPLETED, FAILED, CANCELLED
    status VARCHAR(20) DEFAULT 'PENDING',

    -- 时间戳
    requested_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    sent_at TIMESTAMP,
    completed_at TIMESTAMP,

    -- 发起人
    requested_by INTEGER REFERENCES users(id),

    -- 结果
    photo_count INTEGER DEFAULT 0,
    error_message TEXT,

    INDEX idx_device_status (device_id, status),
    INDEX idx_requested_at (requested_at)
);
```

## 4. API设计

### 4.1 PhotoResource

```java
@Path("/private/photos")
@Api(tags = {"Photo Upload"})
public class PhotoResource {

    // 请求设备拍照
    @POST
    @Path("/request/{deviceId}")
    public Response requestPhotoUpload(
        @PathParam("deviceId") Integer deviceId,
        PhotoRequest request
    );

    // 获取所有照片
    @GET
    @Path("/all")
    public Response getAllPhotos(
        @QueryParam("page") Integer page,
        @QueryParam("size") Integer size,
        @QueryParam("sort") String sort
    );

    // 获取设备照片
    @GET
    @Path("/device/{deviceId}")
    public Response getPhotosByDevice(
        @PathParam("deviceId") Integer deviceId,
        @QueryParam("from") Long fromTime,
        @QueryParam("to") Long toTime
    );

    // 搜索照片
    @GET
    @Path("/search")
    public Response searchPhotos(
        @QueryParam("keyword") String keyword,
        @QueryParam("deviceId") Integer deviceId,
        @QueryParam("from") Long fromTime,
        @QueryParam("to") Long toTime,
        @QueryParam("tags") String tags
    );

    // 获取单张照片
    @GET
    @Path("/{id}")
    public Response getPhoto(@PathParam("id") Integer id);

    // 更新照片信息
    @PUT
    public Response updatePhoto(Photo photo);

    // 删除照片
    @DELETE
    @Path("/{id}")
    public Response deletePhoto(@PathParam("id") Integer id);

    // 下载照片
    @GET
    @Path("/download/{id}")
    @Produces(MediaType.APPLICATION_OCTET_STREAM)
    public Response downloadPhoto(@PathParam("id") Integer id);

    // 批量下载
    @POST
    @Path("/download/batch")
    @Produces("application/zip")
    public Response downloadPhotosBatch(List<Integer> photoIds);

    // 上传照片（设备端调用）
    @POST
    @Path("/upload")
    @Consumes(MediaType.MULTIPART_FORM_DATA)
    public Response uploadPhoto(
        @FormDataParam("file") InputStream fileInputStream,
        @FormDataParam("file") FormDataContentDisposition fileMetaData,
        @FormDataParam("metadata") String metadata
    );

    // 获取上传请求列表
    @GET
    @Path("/requests")
    public Response getPhotoRequests(
        @QueryParam("deviceId") Integer deviceId,
        @QueryParam("status") String status
    );
}
```

### 6.2 PhotoServiceDAO

```java
@Singleton
public class PhotoServiceDAO {
    private final PhotoMapper photoMapper;
    private final PhotoRequestMapper requestMapper;
    private final PushService pushService;
    private final FileStorageService fileStorage;

    @Inject
    public PhotoServiceDAO(
        PhotoMapper photoMapper,
        PhotoRequestMapper requestMapper,
        PushService pushService,
        FileStorageService fileStorage
    ) {
        this.photoMapper = photoMapper;
        this.requestMapper = requestMapper;
        this.pushService = pushService;
        this.fileStorage = fileStorage;
    }

    @Transactional
    public PhotoRequest requestPhotoUpload(Integer deviceId, PhotoRequest request) {
        User currentUser = SecurityContext.get().getCurrentUser().get();

        request.setDeviceId(deviceId);
        request.setCustomerId(currentUser.getCustomerId());
        request.setRequestedBy(currentUser.getId());
        request.setStatus("PENDING");
        request.setRequestedAt(System.currentTimeMillis());

        requestMapper.insertRequest(request);

        // 发送拍照请求推送
        sendPhotoRequest(deviceId, request);

        return request;
    }

    private void sendPhotoRequest(Integer deviceId, PhotoRequest request) {
        ObjectNode payload = JsonMapper.createObjectNode();
        payload.put("requestId", request.getId());
        payload.put("cameraType", request.getCameraType());
        payload.put("resolution", request.getResolution());
        payload.put("includeLocation", request.getIncludeLocation());
        payload.put("autoUpload", request.getAutoUpload());
        payload.put("timeout", 300); // 5分钟超时

        boolean sent = pushService.sendMessage(
            deviceId,
            PushMessage.TYPE_PHOTO_UPLOAD_REQUEST,
            payload.toString()
        );

        if (sent) {
            requestMapper.updateStatus(request.getId(), "SENT", null);
        }
    }

    @Transactional
    public Photo savePhoto(Photo photo, InputStream fileStream) {
        // 生成文件名
        String fileName = generateFileName(photo);
        String filePath = buildFilePath(photo.getDeviceId(), fileName);

        // 保存文件
        long fileSize = fileStorage.store(filePath, fileStream);

        // 获取图片尺寸
        ImageDimensions dimensions = ImageUtils.getDimensions(fileStorage.getFile(filePath));

        photo.setFileName(fileName);
        photo.setFilePath(filePath);
        photo.setFileSize(fileSize);
        photo.setWidth(dimensions.getWidth());
        photo.setHeight(dimensions.getHeight());
        photo.setMimeType(dimensions.getMimeType());
        photo.setUploadedAt(System.currentTimeMillis());

        photoMapper.insertPhoto(photo);

        // 更新请求状态
        if (photo.getRequestId() != null) {
            requestMapper.incrementPhotoCount(photo.getRequestId());
            requestMapper.updateStatus(photo.getRequestId(), "COMPLETED", null);
        }

        return photo;
    }

    public InputStream getPhotoStream(Integer photoId) {
        Photo photo = photoMapper.getPhotoById(photoId);
        if (photo == null || photo.getIsDeleted()) {
            return null;
        }
        return fileStorage.getFile(photo.getFilePath());
    }

    private String generateFileName(Photo photo) {
        String timestamp = new SimpleDateFormat("yyyyMMdd_HHmmss")
            .format(new Date(photo.getTakenAt()));
        return String.format("IMG_%s_%s_%d.jpg",
            timestamp, photo.getCameraType(), photo.getDeviceId());
    }

    private String buildFilePath(Integer deviceId, String fileName) {
        String datePath = new SimpleDateFormat("yyyy/MM")
            .format(new Date());
        return String.format("photos/%d/%s/%s", deviceId, datePath, fileName);
```

```java
// PhotoCaptureService.java
public class PhotoCaptureService extends Service {
    private CameraDevice cameraDevice;
    private ImageReader imageReader;
    private LocationManager locationManager;

    @Override
    public int onStartCommand(Intent intent, int flags, int startId) {
        int requestId = intent.getIntExtra("requestId", -1);
        String cameraType = intent.getStringExtra("cameraType");
        String resolution = intent.getStringExtra("resolution");
        boolean includeLocation = intent.getBooleanExtra("includeLocation", true);

        // 更新请求状态
        updateRequestStatus(requestId, "CAPTURING");

        // 打开相机拍照
        openCamera(cameraType, resolution, (image) -> {
            // 获取位置
            Location location = includeLocation ? getLastLocation() : null;

            // 保存并上传
            saveAndUploadPhoto(image, location, requestId);
        });

        return START_NOT_STICKY;
    }

    private void openCamera(String cameraType, String resolution, PhotoCallback callback) {
        CameraManager cameraManager = (CameraManager) getSystemService(Context.CAMERA_SERVICE);

        try {
            String cameraId = getCameraId(cameraType);

            // 设置分辨率
            Size targetSize = getResolutionSize(resolution);
            imageReader = ImageReader.newInstance(
                targetSize.getWidth(),
                targetSize.getHeight(),
                ImageFormat.JPEG,
                2
            );

            imageReader.setOnImageAvailableListener(reader -> {
                Image image = reader.acquireLatestImage();
                callback.onPhotoCaptured(image);
                image.close();
            }, null);

            // 打开相机
            cameraManager.openCamera(cameraId, new CameraDevice.StateCallback() {
                @Override
                public void onOpened(CameraDevice camera) {
                    cameraDevice = camera;
                    createCaptureSession();
                }

                @Override
                public void onError(CameraDevice camera, int error) {
                    Log.e(TAG, "Camera error: " + error);
                }
            }, null);

        } catch (CameraAccessException e) {
            Log.e(TAG, "Failed to open camera", e);
        }
    }

    private void createCaptureSession() {
        try {
            List<Surface> outputs = Arrays.asList(imageReader.getSurface());

            cameraDevice.createCaptureSession(outputs, new CameraCaptureSession.StateCallback() {
                @Override
                public void onConfigured(CameraCaptureSession session) {
                    try {
                        CaptureRequest.Builder builder = cameraDevice.createCaptureRequest(
                            CameraDevice.TEMPLATE_STILL_CAPTURE
                        );
                        builder.addTarget(imageReader.getSurface());

                        // 自动对焦
                        builder.set(CaptureRequest.CONTROL_AF_MODE,
                            CaptureRequest.CONTROL_AF_MODE_CONTINUOUS_PICTURE);

                        // 自动曝光
                        builder.set(CaptureRequest.CONTROL_AE_MODE,
                            CaptureRequest.CONTROL_AE_MODE_ON_AUTO_FLASH);

                        session.capture(builder.build(), null, null);

                    } catch (CameraAccessException e) {
                        Log.e(TAG, "Failed to capture", e);
                    }
                }
            }, null);

        } catch (CameraAccessException e) {
            Log.e(TAG, "Failed to create session", e);
        }
    }

    private void saveAndUploadPhoto(Image image, Location location, int requestId) {
        updateRequestStatus(requestId, "UPLOADING", null);

        // 将Image转换为JPEG字节数组
        byte[] jpegData = imageToJpeg(image);

        // 构建元数据
        JSONObject metadata = new JSONObject();
        try {
            metadata.put("requestId", requestId);
            metadata.put("takenAt", System.currentTimeMillis());
            metadata.put("cameraType", getCurrentCameraType());

            if (location != null) {
                metadata.put("latitude", location.getLatitude());
                metadata.put("longitude", location.getLongitude());
            }
        } catch (JSONException e) {
            Log.e(TAG, "Failed to build metadata", e);
        }

        // 上传到服务器
        ServerConnection.uploadPhoto(jpegData, metadata.toString(),
            new UploadCallback() {
                @Override
                public void onSuccess() {
                    updateRequestStatus(requestId, "COMPLETED", null);
                }

                @Override
                public void onError(String error) {
                    updateRequestStatus(requestId, "FAILED", error);
                }
            }
        );
    }

    private void updateRequestStatus(int requestId, String status, String error) {
        JSONObject statusUpdate = new JSONObject();
        try {
            statusUpdate.put("requestId", requestId);
            statusUpdate.put("status", status);
            if (error != null) {
                statusUpdate.put("error", error);
            }
            ServerConnection.sendPhotoRequestStatus(statusUpdate);
        } catch (JSONException e) {
            Log.e(TAG, "Failed to update status", e);
        }
    }

    // 远程命令接收器
    public static class PhotoRequestReceiver extends BroadcastReceiver {
        @Override
        public void onReceive(Context context, Intent intent) {
            if (PushMessage.TYPE_PHOTO_UPLOAD_REQUEST.equals(
                intent.getStringExtra("messageType"))) {

                String payload = intent.getStringExtra("payload");
                try {
                    JSONObject request = new JSONObject(payload);

                    Intent serviceIntent = new Intent(context, PhotoCaptureService.class);
                    serviceIntent.putExtra("requestId", request.getInt("requestId"));
                    serviceIntent.putExtra("cameraType", request.optString("cameraType", "BACK"));
                    serviceIntent.putExtra("resolution", request.optString("resolution", "MEDIUM"));
                    serviceIntent.putExtra("includeLocation", request.optBoolean("includeLocation", true));

                    context.startService(serviceIntent);

                } catch (JSONException e) {
                    Log.e(TAG, "Failed to parse request", e);
                }
            }
        }
    }
}
```

```java
        @FormDataParam("file") FormDataContentDisposition fileMetaData,

        try {
            JSONObject metadata = new JSONObject(metadataJson);

            Photo photo = new Photo();
            photo.setDeviceId(metadata.getInt("deviceId"));
            photo.setCustomerId(getCustomerIdFromDevice(metadata.getInt("deviceId")));
            photo.setTakenAt(metadata.optLong("takenAt", System.currentTimeMillis()));
            photo.setCameraType(metadata.optString("cameraType", "BACK"));

            if (metadata.has("latitude") && metadata.has("longitude")) {
                photo.setLatitude(BigDecimal.valueOf(metadata.getDouble("latitude")));
                photo.setLongitude(BigDecimal.valueOf(metadata.getDouble("longitude")));
            }

            if (metadata.has("requestId")) {
                photo.setRequestId(metadata.getInt("requestId"));
            }

            Photo saved = photoDAO.savePhoto(photo, fileInputStream);

            return Response.ok(saved).build();

        } catch (Exception e) {
            return Response.status(Response.Status.INTERNAL_SERVER_ERROR)
                .entity("Upload failed: " + e.getMessage())
                .build();
        }
    }
```

## 7. 安全考虑

1. **隐私保护**: 拍照前需确保符合当地法律法规和公司政策
2. **数据加密**: 照片上传使用HTTPS加密传输
3. **访问控制**: 只有授权管理员可以查看设备照片
4. **审计日志**: 记录所有拍照请求和访问日志
5. **存储安全**: 照片文件存储在安全目录，防止未授权访问
6. **数据保留**: 支持设置照片自动删除策略
