# Feature 6: 网络流量过滤 (Network Traffic Filtering)

## 1. 功能概述

网络流量过滤功能允许管理员配置设备的网络访问规则，控制应用的网络访问权限，实现应用级别的防火墙功能。

### 核心功能
- 应用网络访问控制（允许/阻止）
- URL/域名黑白名单
- IP地址过滤
- 时间段控制
- WiFi/移动数据分别控制
- VPN集成

## 3. 数据库设计

### 3.1 防火墙规则表 (firewall_rules)

```sql
CREATE TABLE firewall_rules (
    id SERIAL PRIMARY KEY,
    configuration_id INTEGER REFERENCES configurations(id) ON DELETE CASCADE,
    customer_id INTEGER NOT NULL,

    -- 规则名称
    rule_name VARCHAR(255) NOT NULL,
    description TEXT,

    -- 规则类型: APP, URL, IP
    rule_type VARCHAR(20) NOT NULL,

    -- 动作: ALLOW, BLOCK
    action VARCHAR(20) DEFAULT 'BLOCK',

    -- 优先级 (数值越小优先级越高)
    priority INTEGER DEFAULT 100,

    -- 目标 (应用包名/URL/IP)
    target VARCHAR(500) NOT NULL,

    -- 网络类型: WIFI, MOBILE, BOTH
    network_type VARCHAR(20) DEFAULT 'BOTH',

    -- 时间控制
    time_start TIME,
    time_end TIME,
    days_of_week VARCHAR(20), -- 1,2,3,4,5 (周一到周五)

    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    INDEX idx_config_type (configuration_id, rule_type),
    INDEX idx_priority (priority)
);
```

## 6. 关键代码示例

### 6.1 Android Agent端流量过滤

```java
// TrafficFilterService.java
public class TrafficFilterService extends VpnService {
    private ParcelFileDescriptor vpnInterface;
    private FirewallConfig config;

    @Override
    public int onStartCommand(Intent intent, int flags, int startId) {
        if (intent != null && "UPDATE_CONFIG".equals(intent.getAction())) {
            config = loadConfigFromIntent(intent);
            applyFirewallRules();
        }
        return START_STICKY;
    }

    private void applyFirewallRules() {
        Builder builder = new Builder();

        // 配置VPN接口
        builder.setMtu(1500);
        builder.addAddress("10.0.0.2", 32);
        builder.addRoute("0.0.0.0", 0);
        builder.addDnsServer("8.8.8.8");

        // 应用应用级规则
        for (FirewallRule rule : config.getAppRules()) {
            if ("BLOCK".equals(rule.getAction())) {
                builder.addDisallowedApplication(rule.getTarget());
            } else {
                builder.addAllowedApplication(rule.getTarget());
            }
        }

        // 建立VPN连接
        vpnInterface = builder.establish();

        // 启动数据包处理线程
        startPacketProcessing();
    }

    private void startPacketProcessing() {
        new Thread(() -> {
            FileInputStream in = new FileInputStream(vpnInterface.getFileDescriptor());
            FileOutputStream out = new FileOutputStream(vpnInterface.getFileDescriptor());

            ByteBuffer packet = ByteBuffer.allocate(32767);

            while (true) {
                try {
                    int length = in.read(packet.array());
                    if (length > 0) {
                        packet.limit(length);

                        // 解析数据包
                        if (shouldBlockPacket(packet)) {
                            // 阻止数据包（不转发）
                            logBlockedPacket(packet);
                        } else {
                            // 转发数据包
                            out.write(packet.array(), 0, length);
                        }

                        packet.clear();
                    }
                } catch (IOException e) {
                    Log.e(TAG, "Error processing packet", e);
                }
            }
        }).start();
    }

    private boolean shouldBlockPacket(ByteBuffer packet) {
        // 解析IP头
        int version = (packet.get(0) >> 4) & 0xF;

        if (version == 4) {
            // IPv4
            int protocol = packet.get(9) & 0xFF;
            String dstIp = String.format("%d.%d.%d.%d",
                packet.get(16) & 0xFF,
                packet.get(17) & 0xFF,
                packet.get(18) & 0xFF,
                packet.get(19) & 0xFF);

            // 检查IP黑名单
            for (String blockedIp : config.getBlockedIps()) {
                if (dstIp.equals(blockedIp)) {
                    return true;
                }
            }
        }

        return false;
    }
}
```

## 7. 安全考虑

1. **系统权限**: 需要VPN权限才能进行流量过滤
2. **性能影响**: 流量过滤会影响网络性能，需优化
3. **绕过风险**: 用户可能禁用VPN服务，需要设备Owner权限强制启用
