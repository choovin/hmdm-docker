# Feature 7: LDAP集成 (LDAP Integration)

## 1. 功能概述

LDAP集成功能允许企业使用现有的LDAP/Active Directory服务器进行用户身份验证，实现统一的身份管理。

### 核心功能
- LDAP服务器配置
- 用户身份验证
- 用户同步
- 组映射
- 多LDAP服务器支持
- SSL/TLS加密连接

## 3. 数据库设计

### 3.1 LDAP配置表 (ldap_settings)

```sql
CREATE TABLE ldap_settings (
    id SERIAL PRIMARY KEY,
    customer_id INTEGER NOT NULL,

    -- 服务器配置
    server_url VARCHAR(255) NOT NULL,
    use_ssl BOOLEAN DEFAULT FALSE,
    use_tls BOOLEAN DEFAULT TRUE,
    port INTEGER DEFAULT 389,

    -- 绑定DN
    bind_dn VARCHAR(500),
    bind_password TEXT,

    -- 用户搜索配置
    base_dn VARCHAR(500) NOT NULL,
    user_search_filter VARCHAR(500) DEFAULT "(uid={0})",
    user_search_base VARCHAR(500),

    -- 属性映射
    username_attribute VARCHAR(100) DEFAULT "uid",
    email_attribute VARCHAR(100) DEFAULT "mail",
    first_name_attribute VARCHAR(100) DEFAULT "givenName",
    last_name_attribute VARCHAR(100) DEFAULT "sn",

    -- 组配置
    group_search_base VARCHAR(500),
    group_search_filter VARCHAR(500) DEFAULT "(member={0})",
    group_name_attribute VARCHAR(100) DEFAULT "cn",

    -- 同步配置
    auto_sync BOOLEAN DEFAULT FALSE,
    sync_interval INTEGER DEFAULT 24, -- 小时
    last_sync_at TIMESTAMP,

    -- 启用状态
    is_enabled BOOLEAN DEFAULT FALSE,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    UNIQUE(customer_id)
);
```

## 6. 关键代码示例

### 6.1 LDAP认证服务

```java
@Singleton
public class LdapAuthenticationService {
    private final LdapSettingsMapper ldapSettingsMapper;
    private final UserDAO userDAO;

    public LdapAuthenticationResult authenticate(String username, String password) {
        LdapSettings settings = ldapSettingsMapper.getSettings();

        if (settings == null || !settings.getIsEnabled()) {
            return LdapAuthenticationResult.fail("LDAP not configured");
        }

        Hashtable<String, String> env = new Hashtable<>();
        env.put(Context.INITIAL_CONTEXT_FACTORY, "com.sun.jndi.ldap.LdapCtxFactory");
        env.put(Context.PROVIDER_URL, buildProviderUrl(settings));
        env.put(Context.SECURITY_AUTHENTICATION, "simple");

        try {
            // 使用绑定DN连接
            env.put(Context.SECURITY_PRINCIPAL, settings.getBindDn());
            env.put(Context.SECURITY_CREDENTIALS, decryptPassword(settings.getBindPassword()));

            // 启用TLS
            if (settings.getUseTls()) {
                env.put(Context.SECURITY_PROTOCOL, "ssl");
            }

            DirContext ctx = new InitialDirContext(env);

            // 搜索用户
            String searchFilter = settings.getUserSearchFilter().replace("{0}", username);
            SearchControls searchControls = new SearchControls();
            searchControls.setSearchScope(SearchControls.SUBTREE_SCOPE);

            NamingEnumeration<SearchResult> results = ctx.search(
                settings.getUserSearchBase(),
                searchFilter,
                searchControls
            );

            if (!results.hasMore()) {
                return LdapAuthenticationResult.fail("User not found");
            }

            SearchResult result = results.next();
            String userDn = result.getNameInNamespace();

            // 验证用户密码
            env.put(Context.SECURITY_PRINCIPAL, userDn);
            env.put(Context.SECURITY_CREDENTIALS, password);

            // 尝试使用用户凭据绑定
            new InitialDirContext(env);

            // 提取用户信息
            Attributes attrs = result.getAttributes();
            LdapUserInfo userInfo = extractUserInfo(attrs, settings);

            // 同步用户到本地数据库
            User localUser = syncUserToLocal(userInfo);

            ctx.close();

            return LdapAuthenticationResult.success(localUser);

        } catch (AuthenticationException e) {
            return LdapAuthenticationResult.fail("Invalid credentials");
        } catch (NamingException e) {
            return LdapAuthenticationResult.fail("LDAP error: " + e.getMessage());
        }
    }

    private String buildProviderUrl(LdapSettings settings) {
        String protocol = settings.getUseSsl() ? "ldaps" : "ldap";
        return String.format("%s://%s:%d",
            protocol, settings.getServerUrl(), settings.getPort());
    }

    private LdapUserInfo extractUserInfo(Attributes attrs, LdapSettings settings)
        throws NamingException {

        LdapUserInfo info = new LdapUserInfo();
        info.setUsername(getAttributeValue(attrs, settings.getUsernameAttribute()));
        info.setEmail(getAttributeValue(attrs, settings.getEmailAttribute()));
        info.setFirstName(getAttributeValue(attrs, settings.getFirstNameAttribute()));
        info.setLastName(getAttributeValue(attrs, settings.getLastNameAttribute()));

        return info;
    }

    private User syncUserToLocal(LdapUserInfo ldapUser) {
        User existing = userDAO.findByUsername(ldapUser.getUsername());

        if (existing == null) {
            // 创建新用户
            User newUser = new User();
            newUser.setLogin(ldapUser.getUsername());
            newUser.setEmail(ldapUser.getEmail());
            newUser.setName(ldapUser.getFirstName() + " " + ldapUser.getLastName());
            newUser.setAuthMethod("LDAP");
            newUser.setPassword("LDAP_AUTHENTICATED"); // 标记为LDAP认证

            return userDAO.insert(newUser);
        } else {
            // 更新用户信息
            existing.setEmail(ldapUser.getEmail());
            existing.setName(ldapUser.getFirstName() + " " + ldapUser.getLastName());
            userDAO.update(existing);
            return existing;
        }
    }
}
```

## 7. 安全考虑

1. **密码加密**: LDAP绑定密码使用加密存储
2. **TLS/SSL**: 强制使用TLS/SSL连接
3. **证书验证**: 验证LDAP服务器证书
4. **密码策略**: 遵循LDAP服务器的密码策略

## 8. 部署说明

1. 执行数据库迁移脚本 `007_ldap_settings.sql`
2. 配置LDAP服务器连接参数
3. 导入LDAP服务器CA证书（如使用自签名证书）
4. 测试连接并同步用户
